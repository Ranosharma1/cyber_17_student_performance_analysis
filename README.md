#  Student Performance Analysis

> A Python-based data analysis project that explores student performance using data visualization and exploratory data analysis (EDA).

---

## About the Project

This project was created to better understand the factors that influence student academic performance. Using Python and data analysis libraries, I explored the dataset, cleaned the data, created visualizations, and identified trends that can help educators and students make better decisions.

The goal wasn't just to create graphs, but to understand the story hidden inside the data.

---

##  What This Project Does

- Cleans and preprocesses the dataset
- Explores the data using Exploratory Data Analysis (EDA)
- Finds relationships between different student attributes
- Visualizes trends using charts and graphs
- Identifies factors that affect student performance
- Generates meaningful insights from the dataset

---

##  Key Insights

Some interesting observations from the analysis include:

- Students with consistent study habits generally perform better.
- Attendance and academic scores show a positive relationship.
- Certain subjects have a stronger impact on overall performance.
- Data visualization makes hidden trends much easier to understand.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Ranosharma1/cyber_17_student_performance_analysis.git
```

Move into the project directory:

```bash
cd cyber_17_student_performance_analysis
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the notebook:

```bash
jupyter notebook
```

Open the notebook and explore the analysis.
